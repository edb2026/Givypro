🛒 Facebook Commerce Export

How it works - one-time setup:
  1. Go to business.facebook.com → create Commerce Manager catalog (type: E-commerce).
  2. Click "Export Images ZIP" below → unpack the archive → upload the images/ folder to your GitHub repo (next to shop.html).
  3. Click "Export Catalog Feed" → upload feed.csv to the same repo.
  4. In Commerce Manager → Data Sources → Add Feed → paste URL [your-site]/feed.csv → set "Update daily".
Every time you add/edit products: just re-export both files and upload them. Facebook re-reads the feed automatically.
